# Secondary Self-Type Relay Lemma

Дата: 2026-05-24.

## Статус

Кандидат-лемма подтверждена на текущем открытом узле:

```text
case45
7*0=4
6*6=8
6*5=7
u=7*6=6*0=2
q=2*6=2
```

Итог:

```text
u=2, q=2 -> closed
```

После этого весь узел

```text
6*6=8
6*5=7
```

закрыт: ранее были закрыты `u=1`; теперь закрыты `u=2` и `u=3`.

## Ролевая форма

Если появляется вторичный self-type:

```text
b3*0=u
u*b3=u
```

то прямое продолжение по старой строке `b3` уже не является главным
структурным объектом. Валидный релейный выход:

```text
s=b3*u
h=s*b3
u*h=0
```

Затем новое ребро `u*h=0` дает следующий релейный слой:

```text
r=u*0
g=r*u
0*g=h
```

Так как строка `0` является полным циклом, `0*g=h` означает:

```text
g=pred0(h)
```

где `pred0(h)` - предшественник `h` в цикле строки `0`.

## Текущие числа

В текущем узле:

```text
b3=6
u=2
```

Поэтому схема имеет вид:

```text
s=6*2
h=s*6
2*h=0
r=2*0
r*2=pred0(h)
```

Важно: недоказанная связь `3*0=h` не используется.

## Проверенный разбор

Перед этим было валидно перепроверено:

```text

```

Оставшиеся значения:

```text

```

закрылись новым релейным слоем.

Для каждого `h` был взят:

```text
r=2*0
r*2=g
```

Все `18` допустимых пар `(h,r)` дали `status: none`.

### `s=3`

### `s=4`

## Структурный вывод

Вторичный self-type не надо добивать старым хвостом строки `6`. Он закрывается
после переноса анализа в строку `u` и затем в цикл строки `0`:

```text
b3*0=u
u*b3=u
=> s=b3*u, h=s*b3, u*h=0
=> r=u*0, r*u=pred0(h)
```

В текущем хвосте это уже не эвристика, а закрывающий слой.

