# Row-6 Cross-Pair Transfer Lemma

Status:

```text
historical transfer note
```

This note records a cross-pair transfer used in row-6 branches.

## Pattern

A forced pair of row-6 values created pressure in a second row.  The second
row then forced a return or collision.

Schematic form:

```text
row 6 forces a pair
  -> E677 creates a predecessor expression
  -> another row receives the pressure
  -> the branch closes or descends
```

## Current role

The value of this note is methodological: it shows how a local pair can become
a two-row pressure configuration.  The current pressure-diamond frontier is a
symbolic version of the same theme.
