Date: 2026-06-17.

```text
candidate / current reduced form of No-Free-Tail
```

## Purpose

The recent branch-closure work has removed the purely local escape cases.

The remaining No-Free-Tail problem is no longer:

```text
classify an isolated first merge or a clean theta endpoint.
```

It is now:

```text
prove termination of the recursive relay among triple fans and mixed 2+1
junctions.
```

## Remaining Relay-Termination Statement

The needed final statement can be written as:

```text
In a finite E677 magma, a relay chain generated by:

1. target swap of a triple fan or mixed 2+1 junction;
2. first-merge target-swap relay;
3. pure-incoming target-swap fan relay;
4. side-attachment mixed-junction relay;
5. minority core-return relay;

cannot continue forever without producing one of:

A. the same ordered two-step interval in two distinct source rows;
B. a right fixer Y*b=b;
C. a strict clean theta, which is already impossible.
```

Item A contradicts two-step source reconstruction.

Item B gives E255 for `b` via:

```text
right_fixer_to_balanced_witness_lemma.md
```

Item C is excluded by:

```text
strict_clean_theta_exclusion_lemma.md
```

## Why This Is Sharper Than The Old No-Free-Tail Form

Earlier, the frontier was a broad branch-closure problem:

```text
two branches born from a fan cannot close freely.
```

Now all one-step local closures have been classified.  The only missing part
is a finite descent or recurrence argument for the relay chain.

The bridge-pair transport lemma gives the likely invariant:

```text
each directed path segment transports bridge pairs
(pred_b(x), pred_x(b)).
```

Thus a relay chain should be tracked by:

```text
target pair,
split vertex,
merge/return vertex,
two bridge labels pred_b(v), pred_v(b),
and the ordered two-step intervals carried by source rows.
```

## Next Proof Target

Find a well-founded measure or recurrence invariant for relay chains.

A plausible measure is not ordinary graph distance, because target swaps can
move the active target from `b` to another vertex.  Bridge-pair states are a
useful guide, but they are not strong enough by themselves.  The source row is
reconstructed only from the full ported interval:

```text
(target, input vertex, output vertex).
```

This correction is recorded in:

```text
ported_interval_state_lemma.md
```

The next task is to prove that a finite relay chain either repeats such a
ported interval state or produces:

```text
a right fixer.
```

This is the current precise frontier for proving No-Free-Tail via the
double-interval pressure mechanism.

## Updated Boundary: Canonical Witness Is Not Enough

The two-edge right-fixer candidate:

```text
Y=(b*c)*(u*k)
```

now appears to collapse to the canonical E255 term:

```text
Y=((b*b)*b)
```

on both `M496` and `F7 x M496` samples.  This is recorded in:

```text
directed_two_edge_canonical_collapse_boundary.md
```

Therefore the main No-Free-Tail proof should not rely on this candidate as a
black-box stopper.  If `Y` is canonical, then proving `Y*b=b` is simply the
original E255 target for `b`.

The cleaner termination route is now:

```text
relay step
=> target-advance of a full ported interval
=> finite recurrence of a full ported interval
=> either cross-role collision or same-row recurrence boundary.
```

The target-advance bookkeeping is recorded in:

```text
ported_interval_state_lemma.md
ported_interval_recurrence_boundary.md
```
