# U-Transfer No-Tail Trap Lemma

Status:

```text
historical candidate / predecessor of the no-free-tail strategy
```

This lemma records an earlier version of the no-tail idea.

## Idea

A `u`-transfer branch looked dangerous when it seemed to create a fresh tail.
The useful observation was that the transfer usually forced one of the
following outcomes:

```text
zero hit
return to an occupied value
short forced cycle
collision with another row output
descent to an older branch
```
