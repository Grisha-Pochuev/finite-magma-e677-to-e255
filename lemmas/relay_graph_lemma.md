# Relay Graph Lemma

Status:

```text
working structural lemma
```

This note summarizes the relay-graph viewpoint used in the finite branch
analysis.

## Idea

Forced edges generated by `E677` can be viewed as a directed relay graph.
Vertices are table values or symbolic expressions.  Edges represent forced row
transitions.

The important question is not merely whether a branch has many forced cells.
The important question is whether those forced cells form one of the following
patterns:

```text
zero hit
return to an occupied row
short forced cycle
collision with an earlier forced output
fresh tail
```

## Relevance

The relay-graph viewpoint led to the current no-free-tail target: in a finite
E677 magma, a fresh tail created by the bad-cycle construction should not be
able to continue indefinitely.
