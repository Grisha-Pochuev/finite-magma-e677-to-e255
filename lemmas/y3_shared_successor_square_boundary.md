## General Square

Fix any product:

```text
r*T=O.
```

It creates two natural second steps:

```text
left-row step:       U=r*O,
fixed-target step:   V=O*T.
```

So:

```text
T -> O -> U       under row r,
r -> O -> V       under right multiplication by T.
```

E677 also gives the return edge:

```text
O*(U*r)=T.
```

Therefore in `H_T` the old source row `r` and the successor source row `O`
give:

```text
row r: P -> O,        where P=T*(O*r)=pred_r(T),
row O: U*r -> V,      where O*(U*r)=T.
```

This is the fixed-target source-successor step.
