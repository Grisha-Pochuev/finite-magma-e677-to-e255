# E677 -> E255 finite magmas

Дата: 2026-05-24.

Этот проект содержит исследование открытой задачи:

```text
E677: x = y * (x * ((y * x) * y))
E255: x = ((x*x)*x)*x
```

Главный вопрос: влечет ли `E677` формулу `E255` для конечных магм.

## Читать сначала

1. `CONTINUE_HERE.md` - текущая точка продолжения. Начинать всегда отсюда.
2. `LEMMA_STATUS.md` - карта статусов лемм: что доказано, что локально
   закрыто, что только кандидат, что было исправлено аудитом.
3. `AGENT_WORKFLOW.md` - правила работы агента, включая запрет на механическое
   добивание веток без новой структуры.
4. `PROJECT_STATUS.md` - широкий статус проекта.
5. `RESULTS_INDEX.md` - навигация по файлам.

## Текущее правило осторожности

Не читать `research_log.md` как текущий статус. Это полный исторический журнал,
в котором есть старые таймауты, старые continuation points и исправленные
гипотезы.

Актуальный порядок:

```text
CONTINUE_HERE.md -> LEMMA_STATUS.md -> нужный файл леммы -> research_log.md точечно
```

Файлы в `archive/` являются историческими и не должны использоваться как
текущий статус.
