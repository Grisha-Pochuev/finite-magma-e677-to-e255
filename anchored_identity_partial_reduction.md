# Anchored Identity Partial Reduction

Date: 2026-06-21.

Status:

```text
partial lemma / reduction target for shared-step anchored identity
```

## Setup

Use the shared-step anchored triangle notation:

```text
p*b=q*b=z,
U=p*z,
W=q*z,
h=U*p=W*q,
z*h=b.
```

Define:

```text
T=U*h,
S=W*h.
```

The strong target is:

```text
T=S.
```

## Standard Consequences Already Available

The usual finite E677 collision consequence gives:

```text
p*b=q*b=z
=> (p*z)*p=(q*z)*q.
```

So:

```text
U*p=W*q=h.
```

The standard predecessor consequence gives:

```text
z*h=b.
```

## New Useful Reduction

Apply the finite E677 consequence:

```text
if a=y*x and c=y*a, then a*(c*y)=x.
```

First use `y=U`, `x=p`:

```text
a=U*p=h,
c=U*h=T,
therefore h*(T*U)=p.
```

Then use `y=W`, `x=q`:

```text
a=W*q=h,
c=W*h=S,
therefore h*(S*W)=q.
```

Thus the shared step can be rewritten as:

```text
(h*(T*U))*b = (h*(S*W))*b = z.
```

This is the current sharper form of the negated strong target:

```text
T!=S
but row h sends T*U to p and S*W to q,
and rows p,q then share the same b-output z.
```

## Interpretation

The strong identity is not yet proved by the shallow local formulas alone.
However, the negation is no longer just three unrelated edges in `H_h`.
It forces a back-projected shared step through the single row `h`:

```text
T*U -> p -> z,
S*W -> q -> z.
```

So the next structural question is:

```text
Can a clean shared-step pair p*b=q*b=z be generated by one row h from
the two inputs T*U and S*W while T=U*h and S=W*h remain distinct?
```

If this is impossible, it proves the strong identity `U*h=W*h`.

If it is possible, the correct residual is not the old X3 route verbatim.
It is an anchored back-projection residual with one row `h`, two generated
inputs `T*U` and `S*W`, and a common old output `z` after multiplying by `b`.
