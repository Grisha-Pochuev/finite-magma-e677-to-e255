## R-b4 Reduction

R-b4 is a first repeat of the right-`b` orbit:

```text
x_i=x_j=v,
0<i<j.
```

It creates an incoming merge at `v` in `H_b`.

Because:

```text
A_i=pred_v(b),
v*A_i=b,
```

there is an automatic outgoing continuation:

```text
v -> A_i*b
```

in `H_b`.

Therefore R-b4 is a first-merge target-swap relay.  After swapping target
`b -> v`, it becomes:

```text
mixed 2+1 junction,
or outgoing triple fan.
```

So R-b4 is absorbed into the standard global relay frontier.

Moreover, if the repeat is internal:

```text
x_i=x_j, 0<i<j,
```

then the closed right-`b` cycle:

```text
x_i -> ... -> x_{j-1} -> x_i
```

has strictly smaller footprint than the full old right-`b` first-repeat
segment:

```text
x_0 -> ... -> x_{j-1}.
```

If `i=0`, the repeat returns to the original crossed-fan vertex and is a
visible/original fan attachment.

## R-b5 Reduction

R-b5 is a row-`b` A-layer cycle:

```text
b*A_j=A_i.
```

At the generated input `A_j`, this is exactly the unavoidable base bridge:

```text
row b:   A_j -> A_i,
row x_j: A_j -> b.
```

So every R-b5 edge is a base row-b/generated bridge whose row-`b` output hits
the generated A-layer.

An R-b5 cycle is therefore a closed chain of base-bridge A-hits:

```text
D_j=A_i.
```

It is not an independent fresh recurrence layer.

An internal first repeat in the A-layer gives a smaller A-layer footprint.
Only the start-return closed A-cycle remains as a minimal R-b5 recurrence
boundary.
